\doxysubsection{Detailed Description}
La classe Ovo\+Loader gestisce il parsing dei file .ovo, estraendo e gestendo i dati della scena 3D. 

La classe Ovo\+Loader si occupa del parsing dei file .ovo, che contengono informazioni sulle scene 3D, inclusi materiali, texture e la struttura gerarchica della scena. Fornisce metodi per il caricamento, l\textquotesingle{}elaborazione e l\textquotesingle{}accesso ai dati analizzati.

\begin{DoxyNote}{Note}
Attualmente, non tutte le informazioni nel file .ovo vengono analizzate o utilizzate; alcuni dati vengono ignorati e non elaborati.
\end{DoxyNote}
La classe include funzionalità per il parsing di materiali, texture, nodi, mesh, mesh con ossatura, ossa e proprietà fisiche. Costruisce una gerarchia di scene rappresentata dalla classe \doxylink{class_node}{Node}, con materiali e texture gestiti tramite puntatori condivisi. La scena analizzata può essere accessa attraverso il nodo radice.

Esempio di utilizzo\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{OvoLoader\ loader;}
\DoxyCodeLine{loader.loadOvoFile(\textcolor{stringliteral}{"{}esempio.ovo"{}});}
\DoxyCodeLine{std::unique\_ptr<Node>\ radiceScena\ =\ std::move(loader.getRootNode());\ \textcolor{comment}{//\ Nota:\ acquisisce\ la\ proprietà del\ nodo\ radice}}

\end{DoxyCode}


\begin{DoxyNote}{Note}
La classe non è ancora completamente implementata, e alcuni dati potrebbero non essere elaborati nella versione attuale. Aggiornamenti futuri potrebbero estendere la funzionalità per gestire ulteriori caratteristiche presenti nel formato del file .ovo. 
\end{DoxyNote}
