\doxysubsection{Detailed Description}
Rappresenta il motore grafico 3D. 

La classe \doxylink{class_engine}{Engine} rappresenta un motore grafico 3D che gestisce la visualizzazione e la manipolazione di oggetti 3D in una scena virtuale.
