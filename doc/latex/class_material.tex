\doxysubsection{Detailed Description}
Rappresenta un materiale in una scena 3D. 

La classe \doxylink{class_material}{Material} rappresenta un materiale utilizzato per la resa di oggetti in una scena 3D. Deriva dalla classe \doxylink{class_object}{Object} e fornisce funzionalità per la gestione delle impostazioni del materiale e delle texture.
