Gestisce il movimento di una \doxylink{class_mesh}{Mesh} selezionata. 

La classe \doxylink{class_mesh_movement_manager}{Mesh\+Movement\+Manager} gestisce il movimento della \doxylink{class_mesh}{Mesh} attiva (selezionata), includendo blinking, velocità e altre configurazioni.
