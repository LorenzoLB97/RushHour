\doxysubsection{Detailed Description}
Rappresenta una sorgente di luce puntiforme nel sistema. 

La classe \doxylink{class_point_light}{Point\+Light} estende la classe base \doxylink{class_light}{Light} e rappresenta una sorgente di luce puntiforme all\textquotesingle{}interno del sistema. La luce punta in tutte le direzioni da una posizione specifica.
