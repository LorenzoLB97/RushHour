\doxysubsection{Detailed Description}
Classe che rappresenta una luce spot nella scena. 

La classe \doxylink{class_spot_light}{Spot\+Light} eredita dalla classe base \doxylink{class_light}{Light} e fornisce funzionalità specifiche per una luce spot, inclusi metodi per impostare la direzione, l\textquotesingle{}angolo di taglio e l\textquotesingle{}esponente della luce spot. La luce spot può essere resa nella scena utilizzando il metodo rendering.

Esempio d\textquotesingle{}uso\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\mbox{\hyperlink{class_spot_light}{SpotLight}}\ mySpotLight(\textcolor{stringliteral}{"{}Spot1"{}});}
\DoxyCodeLine{mySpotLight.setDirection(glm::vec3(1.0f,\ -\/1.0f,\ -\/1.0f));}
\DoxyCodeLine{mySpotLight.setLightCutoff(30.0f);}
\DoxyCodeLine{mySpotLight.spotExponent(50.0f);}

\end{DoxyCode}


\begin{DoxyNote}{Note}
La classe è parte di un sistema di illuminazione più grande e può essere utilizzata per rappresentare una fonte di luce spot nella scena 3D. 
\end{DoxyNote}
