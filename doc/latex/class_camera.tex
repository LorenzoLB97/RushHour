\doxysubsection{Detailed Description}
Rappresenta una telecamera nel sistema. 

La classe \doxylink{class_camera}{Camera} estende la classe \doxylink{class_node}{Node} e rappresenta una telecamera nel sistema. La telecamera può essere configurata per il movimento dinamico o statico, con la possibilità di spostarsi e ruotare in uno spazio tridimensionale.
