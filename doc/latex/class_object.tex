\doxysubsection{Detailed Description}
Rappresenta un oggetto di base all\textquotesingle{}interno del sistema. 

La classe \doxylink{class_object}{Object} fornisce funzionalità di base comuni a tutti gli oggetti nel sistema. Ogni oggetto ha un identificatore univoco, un tipo e un nome associato. La classe è progettata per essere di base per altre classi più specializzate.
