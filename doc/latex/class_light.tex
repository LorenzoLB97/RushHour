\doxysubsection{Detailed Description}
Rappresenta una luce in una scena 3D. 

La classe \doxylink{class_light}{Light} è una classe di base per le luci in una scena 3D. Deriva dalla classe \doxylink{class_node}{Node} e fornisce funzionalità di gestione delle luci come ambient, diffuse e specular lighting.
