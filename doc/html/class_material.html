Detailed Description

Rappresenta un materiale in una scena 3D.

La classe Material rappresenta un materiale utilizzato per la resa di oggetti in una scena 3D. Deriva dalla classe Object e fornisce funzionalità per la gestione delle impostazioni del materiale e delle texture.
