Detailed Description

Rappresenta una telecamera nel sistema.

La classe Camera estende la classe Node e rappresenta una telecamera nel sistema. La telecamera può essere configurata per il movimento dinamico o statico, con la possibilità di spostarsi e ruotare in uno spazio tridimensionale.
