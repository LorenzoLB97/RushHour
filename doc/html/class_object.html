Detailed Description

Rappresenta un oggetto di base all'interno del sistema.

La classe Object fornisce funzionalità di base comuni a tutti gli oggetti nel sistema. Ogni oggetto ha un identificatore univoco, un tipo e un nome associato. La classe è progettata per essere di base per altre classi più specializzate.
