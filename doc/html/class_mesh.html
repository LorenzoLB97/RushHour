Detailed Description

Rappresenta una mesh in una scena 3D.

La classe Mesh rappresenta una mesh utilizzata per la resa di oggetti in una scena 3D. Deriva dalla classe Node e contiene informazioni sul materiale associato e dettagli di livello di dettaglio (LOD).
