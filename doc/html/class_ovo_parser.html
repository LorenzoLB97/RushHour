Detailed Description

La classe OvoLoader gestisce il parsing dei file .ovo, estraendo e gestendo i dati della scena 3D.

La classe OvoLoader si occupa del parsing dei file .ovo, che contengono informazioni sulle scene 3D, inclusi materiali, texture e la struttura gerarchica della scena. Fornisce metodi per il caricamento, l'elaborazione e l'accesso ai dati analizzati.

Note
Attualmente, non tutte le informazioni nel file .ovo vengono analizzate o utilizzate; alcuni dati vengono ignorati e non elaborati.

La classe include funzionalità per il parsing di materiali, texture, nodi, mesh, mesh con ossatura, ossa e proprietà fisiche. Costruisce una gerarchia di scene rappresentata dalla classe Node, con materiali e texture gestiti tramite puntatori condivisi. La scena analizzata può essere accessa attraverso il nodo radice.

Esempio di utilizzo:

OvoLoader loader;
loader.loadOvoFile("esempio.ovo");
std::unique_ptr<Node> radiceScena = std::move(loader.getRootNode()); // Nota: acquisisce la proprietà del nodo radice
Note
La classe non è ancora completamente implementata, e alcuni dati potrebbero non essere elaborati nella versione attuale. Aggiornamenti futuri potrebbero estendere la funzionalità per gestire ulteriori caratteristiche presenti nel formato del file .ovo.

void OvoParser::parseFile ( const std::string & fileName)

Analizza un file .ovo specifico e costruisce il grafico di scena.

Parameters
fileNameIl nome del file .ovo da analizzare, cercato nella directory specificata nel costruttore.

La funzione analizza prima tutto ciò che non è incluso nel grafico di scena (come i materiali), per poi procedere a costruire la scena del grafo basandosi sulle informazioni trovate.
