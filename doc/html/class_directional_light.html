Detailed Description

Rappresenta una luce direzionale in un ambiente grafico 3D.

La classe rappresenta una luce direzionale utilizzata in un ambiente grafico 3D. Estende la classe di base Light e aggiunge funzionalità specifiche per le luci direzionali.
